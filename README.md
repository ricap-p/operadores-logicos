# 🔍 Buscador de Contatos - DevClub

Este é um projeto simples de buscador de contatos desenvolvido com **HTML, CSS e JavaScript**, com foco em praticar manipulação do DOM, lógica de programação e interação com o usuário.

---

## 🚀 Funcionalidades

* 🔎 Buscar contatos pelo nome
* 📋 Exibir nome e telefone do contato
* ⚡ Atualização dinâmica na tela (DOM)
* 🎨 Interface simples e minimalista

---

## 🧠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 💻 Como funciona

O usuário digita o nome de um contato no campo de busca e clica no botão.

O sistema percorre uma lista de contatos (array de objetos) e, se encontrar o nome correspondente, exibe as informações na tela.
